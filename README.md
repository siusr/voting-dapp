# Voting DApp

A simple decentralized voting application built with Solidity, Hardhat, and React.

---

## Tech Stack

- **Solidity**
- **Hardhat**
- **React + ethers.js**
- **MetaMask**

---

## Features

- Create proposals and vote
- Wallet connect (MetaMask)
- Real-time vote count
